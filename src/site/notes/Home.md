## WAF Onboarding Demo
We will be onboarding this site onto the Imperva Cloud WAF during this session. This will involve:
- Creating a configuration on Imperva
- Making a DNS change to validate ownership of the FQDN
- Waiting for the certificate to be issued by Imperva (~5 mins)
- Changing the DNS for the FQDN to point to the Imperva CNAME record
